## Quick Start

Para começar a usar a API Fortuna Pay, siga os passos abaixo:

### 1. Registro

1. Acesse o portal da Fortuna Pay: [https://app.fortunapay.com.br](https://app.fortunapay.com.br)
2. Clique em **Registrar-se** e preencha os dados da sua empresa ou perfil.
3. Envie os documentos solicitados para verificação.

> Observação: Após o registro, sua conta passará por análise interna do time Fortuna Pay. A aprovação dependerá do atendimento aos critérios internos, como conformidade regulatória e dados fornecidos.

### 2. Obtenção do Token

Depois que sua conta for aprovada:

1. Faça login no portal da Fortuna Pay.
2. Gere um **Bearer Token** para autenticação da API.
3. Guarde o token com segurança — ele será necessário em todas as requisições.

### 3. Testando a API

Com o token em mãos, você já pode fazer requisições aos endpoints, como:

- Criar cobrança PIX: `/pix/cob`
- Consultar status de cobrança: `/pix/cob/{txid}`
- Consultar transações: `/pix/transaction/{id}`
