## Introdução

A API da Fortuna Pay permite gerenciar cobranças via PIX, consultar status de transações e integrar pagamentos em sistemas externos de forma segura e rápida.  

Para este guia, todos os endpoints são autenticados usando tokens Bearer.  

## Autenticação

Todos os endpoints exigem **Bearer Token** no header `Authorization`.

```http
Authorization: Bearer SEU_TOKEN_AQUI
Content-Type: application/json
